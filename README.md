# Nodejs hot reloader

### Installation
```
npm i git://github.com:frullah/node-hot-reloader.git -D

or with ssh

npm i git+ssh://github.com:frullah/node-hot-reloader.git -D
```

### Usage
```
hot-reloader <entry-file> [...options]

Options:
  --version    Show version number                                     [boolean]
  -h, --help   Show help                                               [boolean]
  -w, --watch  Watch path target                                        [string]
```
